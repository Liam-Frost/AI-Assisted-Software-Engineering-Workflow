# Decision Index

## Purpose

This document is a **lightweight index of decisions**.

It does NOT contain full decision rationale.
It answers only:

- What was decided
- When
- Where the full record lives

---

## How to Use This File

- Every non-trivial decision should be listed here
- Each entry should link to:
  - an ADR, or
  - a Contract, or
  - an iteration document
- Closed decisions remain listed (do not delete history)

---

## Status Legend

- **Superseded**: replaced by a later decision

# 决策索引（Decision Index）

## 文档目的

本文档是一个**轻量级的决策索引表**。

它不保存完整的决策过程，只回答：

- 做了什么决定
- 什么时候
- 详细记录在哪里

---

## 使用方式

- 所有非琐碎决策都应在此列出
- 每条记录应指向：
  - 对应的 ADR，或
  - Implementation Contract，或
  - 某次迭代文档
- 已失效的决策也应保留（不要删除历史）

---

| D-002 | YYYY-MM-DD | Active | ... | Iteration YYYY-MM-DD |

---

## 状态说明

- **Active**：当前生效
- **Superseded**：已被后续决策替代
- **Deprecated**：仍存在但不推荐继续依赖
